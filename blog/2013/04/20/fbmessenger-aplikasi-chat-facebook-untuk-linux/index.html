FBMESSENGER - Chat Facebook Untuk Linux

Apr 20th, 2013

Bagi pengguna linux, anda mungkin sudah mengenal chat client pidgin yang memudahkan anda untuk selalu terhubung dengan berbagai macam akun chat anda seperti yahoo messenger, google talk, bahkan facebook. Dengan kemudahan seperti ini, anda bisa menyusun seluruh percakapan anda ke dalam sebuah jendela aplikasi. Bagi anda yang lebih menyukai layanan chat yang disediakan oleh facebook, anda mungkin tidak akan menemui fitur chat yang biasa anda gunakan pada aplikasi facebook seperti notikasi, friend request, friend updates dan pop up alert.

Seorang developer facebook yang tentu saja tahu betul fitur-fitur apa saja yang dimiliki facebook chat, membuat sebuah aplikasi chat yang serupa dengan facebook messenger yang biasa anda gunakan. Berikut ini merupakan langkah-langkah untuk menggunakan fbmessenger pada beberapa distribusi linux.

  • Bagi pengguna Ubuntu Linux, tambahkan repositori di bawah ini

Update, kemudian instal fbmessenger

Tampilan aplikasi fbmessenger pada ubuntu linux

  • Bagi pengguna Arch Linux menggunakan AUR melalui tautan ini
  • Bagi distribusi linux yang lain, bisa menggunakan source code fbmessenger yang telah disertakan dengan skrip untuk membuat paket rpm & debian melaui github di tautan ini
