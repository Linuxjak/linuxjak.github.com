Debian 7 - Kau Bukan Yang Dulu Lagi

May 8th, 2013

Berbicara mengenai Debian Linux, secara pribadi jika seseorang menanyakan kepada saya mengapa menggunakan Debian Linux, adalah karena distribusi linux inilah yang saya gunakan pertama kali sebagai sistim operasi server. Keinginan menggunakan linux sebagai sistim operasi yang terkenal paling handal di kelas server membuat saya memilih Debian karena kemudahan untuk menggunakannya. Saya waktu itu hampir tidak percaya bahwa hanya dengan perintah apt-get dan sedikit saja konfigurasi, saya bisa membangun sebuah web server, lengkap dengan ftp server, php dan database mysql yang saya butuhkan.

Berawal dari Debian versi ke 5 kemudian Debian 6, saya sudah merasa nyaman untuk tetap menggunakan sistim operasi ini untuk server. Belum terpikir untuk menggunakannya sebagai sistim operasi desktop, apalagi setelah saya tau begitu ketatnya kontrol pengembang Debian terhadap aplikasi yang dipaketkan ke dalam sistim operasi ini (tentu saja untuk tetap menjaga kestabilan & kehandalannya), jangan harap untuk mendapatkan aplikasi desktop terbaru yang belum melalui tahap ujicoba dan memenuhi beberapa standar perangkat lunak terbuka, sehingga saya lebih memilih Ubuntu sebagai sistim operasi desktop saya agar saya tetap mendapatkan pembaharuan aplikasi desktop yang biasa saya gunakan minimal setiap 6 bulan sekali.

Saya hampir tidak pernah memikirkan pembaharuan dari Debian Linux. Saya pernah merasakan rentang waktu yang cukup lama versi ke 5 Debian untuk melalui proses pembaharuan ke versi ke 6, hingga seorang rekan saya di KPLI Jakarta memberi tau saya Debian versi ke 7 (wheezy) sudah dirilis, langsung saja saya “menuju tkp” untuk mengunduh versi terbaru tersebut, dan setelah melalui proses instalasi, Debian 7 - Kau memang bukan yang dulu lagi.

Perubahan terbesar yang saya lihat adalah adanya dektop manager yang baru seperti gnome 3.4, kde 4.8 dan xfce 4.8 dan aplikasi open office yang sudah diganti dengan libreoffice (walau masih versi 3), di bawah ini beberapa tangkapan layar yang saya ambil menggunakan gnome desktop manager.
