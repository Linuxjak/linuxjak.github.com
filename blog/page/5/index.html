Bercengkrama Dengan Ruby on Rails

Dec 8th, 2012

Apaan sih Ruby on Rails?

Ruby on Rails atau biasa disebut Rails merupakan web framework berbasis Ruby yang dibuat untuk membuat pengembangan aplikasi web menjadi lebih mudah, menyenangkan dan meningkatkan produktivitas melalui mekanisme arsitektur MVC yang digunakan nya.

Apaan sih arsitektur MVC itu?

Model - View - Control atau yang biasa disingkat MVC merupakan sebuah arsitektur pengembangan aplikasi yang memisahkan antara bagian logic aplikasi dengan antar muka aplikasi, dengan demikian aplikasi yang dikembangkan pun akan mudah untuk dipelihara.

Menggunakan Vim Editor

Dec 8th, 2012

Vim merupakan editor standar yang ada pada setiap distribusi sistem operasi Unix/Linux. Berbeda dengan editor lainnya (seperti gedit atau notepad) vim menggunakan konsol / command untuk antar-mukanya. Pekerjaan administrasi sistem bisa dilakukan secara lokal dan remote menggunakan vim, karena itu menguasai editor ini merupakan suatu keharusan, khususnya bagi anda yang harus melakukan konfigurasi sistem atau remote debugging baik secara lokal maupun remote. Seperti telah di singgung di atas,vim menggunakan konsol / command untuk antar mukanya, artinya tidak memerlukan sumber daya komputasi yang besar, sehingga bisa di eksekusi dengan cepat. Artikel kali ini akan membahas beberapa perintah dasar vim yang biasa digunakan
