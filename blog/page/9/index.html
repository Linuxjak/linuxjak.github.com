Mengenal HTML 5

Nov 16th, 2012

HTML 5 adalah versi terbaru dari HTML yang akan menjadi standar baru untuk HTML, XHTML, dan HTML DOM. HTML 5 merupakan kelanjutan dari HTML versi 4 yang diluncurkan pada tahun 1994. Sejak pertama kali diluncurkan, HTML 5 kini sudah didukung oleh sebagian besar browser. HTML 5 sendiri pengembangannya ditujukan untuk mengurangi kebutuhan eksternal plugin seperti flash dan melakukan error handling yang lebih baik dalam mengelola dan menampilkan konten web yang semakin kaya (Rich Internet Application).

HTML 5 memiliki spesifikasi scripting baru yang membedakannya dengN HTML 4, beberapa scripting baru HTML 5 antara lain adalah adanya fitur canvas scripting untuk menggambar, local storage dan scripting yang lebih spesifik terhadap konten seperti footer, header, nav dan action.
